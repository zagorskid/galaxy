Welcome to the Galaxy at University of Silesia!

Galaxy is a very functional and easy to use set of tools for data-intensive biology analyses, equipped with user friendly graphical web interface. This platform allows to process big amounts of genomic data using the power of High Performance Computing Clusters. Obtained results can be easily stored, shared and visualized.

Galaxy is an open-source project. It’s user friendly, web-based platform for accessible, reproducible, and transparent computational biomedical research. With Galaxy users without programming experience can easily specify parameters and run organised workflows and many separate bioinformatic tools. Galaxy is also reproducible, because it captures information so that any user can repeat and understand a complete computational analysis. Moreover, users can easily share and publish analyses via the web and create Pages – interactive, web-based documents that describe a complete analyses process. These workflows are built by connecting the tools available in the Galaxy to automatically run a sequence of steps.
